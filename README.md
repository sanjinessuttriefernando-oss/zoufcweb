# Mi Página de la UFC

Un sitio web estático dedicado al mundo de la Ultimate Fighting Championship (UFC), con noticias, galería de momentos y más.

## Características
- Noticias actualizadas (Blog).
- Galería multimedia con videos de momentos destacados.
- Diseño responsivo con **Bootstrap**.
- Secciones informativas sobre la organización y servicios.

## Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Cómo Visualizarlo
Solo clona el repositorio y abre `index.html` en tu navegador.
